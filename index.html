Objetos JavaScript

Em JavaScript, um objeto é uma coleção de propriedades, e uma propriedade é uma associação entre um nome (ou chave) e um valor. Os valores das propriedades podem ser funções, que são então chamadas de métodos. Objetos podem ser criados usando a notação literal de objetos ou a função construtora Object().

Exemplo de Objeto

Aqui está um exemplo simples de um objeto em JavaScript:

        const pessoa = {
            nome: "João",
            idade: 30,
            cidade: "São Paulo"
        };
